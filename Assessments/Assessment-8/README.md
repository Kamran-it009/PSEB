# 🐍 Python Functions --- Practical Practice Questions

### 1. Student Grade Calculator

Write a function `calculate_grade(marks)` that takes marks as input and
returns grade according to:

-   90 and above → A
-   80 -- 89 → B
-   70 -- 79 → C
-   Below 70 → Fail

------------------------------------------------------------------------

### 2. Even or Odd Number Checker

Write a function `is_even(number)` that returns:

-   `True` if the number is even
-   `False` if the number is odd

------------------------------------------------------------------------

### 3. Factorial Using Function

Write a function `factorial(n)` that returns the factorial of a number
using a loop.

Example:

    factorial(5)
    Output → 120

------------------------------------------------------------------------

### 4. Find Maximum Number

Write a function `find_max(a, b, c)` that returns the largest number
among three numbers.

------------------------------------------------------------------------

### 5. Word Counter

Write a function `count_words(sentence)` that counts how many words
exist in a given sentence.

Example:

    "I love Python programming"
    Output → 4

------------------------------------------------------------------------

### 6. Prime Number Checker

Write a function `is_prime(n)` that checks whether a number is a prime
number.

------------------------------------------------------------------------

### 7. List Sum Function

Write a function `list_sum(numbers)` that takes a list as input and
returns the sum of all numbers.

Example:

    [10, 20, 30]
    Output → 60

------------------------------------------------------------------------

### 8. Password Strength Checker

Write a function `check_password(password)` that checks:

-   Minimum 8 characters required
-   Must contain at least one digit

Return:

-   "Strong" or "Weak"

------------------------------------------------------------------------

### 9. Temperature Converter

Write a function `celsius_to_fahrenheit(c)` that converts temperature
from Celsius to Fahrenheit.

Formula:

    F = (C × 9/5) + 32

------------------------------------------------------------------------

### 10. Remove Duplicates from List

Write a function `remove_duplicates(data)` that removes duplicate
elements from a list and returns a new list.

Example:

    [1,2,2,3,4,4]
    Output → [1,2,3,4]

------------------------------------------------------------------------
